# 关于我

## 个人简介

我是硬件电子工程师，专注于 **消费电源与快充硬件研发**。

多年自主开发实战项目，设计风格注重：

- **工程落地**

- **高稳定性**

- **高兼容性**

- **可量产**

我擅长从零开始完成电源产品开发的全流程：

方案选型 → 原理图设计 → PCB布局 → 仿真验证 → 样机调试 → EMC整改 → 量产优化

## 职业理念

不做理论设计，只做**能稳定工作、能过测试、能量产**的硬件方案。
