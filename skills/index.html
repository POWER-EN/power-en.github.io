一、电路设计能力#

  • 开关电源拓扑：Buck / Boost / Buck-Boost 设计与环路调试

  • USB PD快充协议电路、多品牌快充兼容电路开发

  • 锂电池充放电管理、过充过放短路保护、NTC温控电路

  • 电量计采样、电压电流检测、数模混合控制电路

二、PCB设计能力#

  • 2层/4层高密度PCB Layout

  • 大功率电源散热铺铜、多过孔、热源分区布局

  • 功率地/信号地分割、模拟数字隔离、EMC布局优化

  • DFM可制造性优化，适配SMT批量生产

三、熟练工具#

  • 设计软件：AD、KiCad、立创EDA、3D结构预览

  • 仿真工具：LTspice 电源环路、纹波、补偿仿真

  • 测试设备：示波器、电子负载、可调电源、测温仪器

  • 文档输出：BOM整理、规格书、测试报告、整改复盘文档

四、工程实战能力#

  • 产品方案评估、器件选型、成本优化

  • 样机焊接、上电调试、故障定位

  • 传导辐射EMI整改、温升优化、待机功耗优化

  • 对接打板、贴片、解决量产稳定性问题
